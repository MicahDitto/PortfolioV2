Client Log Monitoring Dashboard

In this project, I Designed and created monitoring dashboards for critical airline cloud systems and applications using Sumologic and Dynatrace log monitoring.

Overview

In the airline industry, ensuring uninterrupted performance and pinpointing potential issues for cloud systems and applications is paramount. This Log Monitoring Dashboard was my first assignment with this client giving me an opportunity to design and develop an intuitive dashboard for my manager's manager using cutting-edge technologies such as Sumo Logic and Dynatrace log monitoring.

Client Needs

The client, a prominent player in the airline industry, required a robust solution for real-time monitoring and analysis of their cloud systems and applications' log data from SumoLogic and Dynatrace. This was vital for maintaining operational excellence and proactively addressing any performance bottlenecks such as downed servers or apps, security issues, and irregularities in scheduled functions. There were ways to check each of these web apps individually, but the stakeholders needed a way to view their information in a combined view.

Dashboard Design

I meticulously designed the monitoring dashboards, focusing on making the interface user-friendly and intuitive data visualization. The goal was to empower stakeholders to quickly obtain insights from complex log data all in a combined view in PowerBI.

Continuously involving the real users in the design and testing process helped me identify usability issues and gather feedback for improvements. I designed the aesthetic to be dark with an adjusted contrast to reduce eye fatigue considering my team managers frequent late nights. Other considerations in the design process were mobile friendly responsiveness, which I also achieved.

Challenges Faced

  1. User Experience and Data Visualization:

    Designing dashboards that effectively conveyed insights from complex log data to users with varying levels of technical expertise was a design and usability challenge. We needed to ensure that the data was presented in a user-friendly and actionable manner.

      Solutions:

      • Implementing user-centric Information Architecture.
      • Conducting user research interviews and surveys.
      • Making designs consistent with other client platforms.

  2. Integration with Existing Infrastructure:

    Ensuring seamless integration with the client's existing infrastructure, including third-party tools and monitoring systems, required careful planning and compatibility testing.

      Solutions:

      • Collaborating with PBI Consultants
      • Comprehensively learning API documentation.
      • Constant communication with key stakeholders.

  3. Customization Needs:

    Catering to the unique monitoring requirements of different airline systems and applications called for a high degree of customization. Balancing customization with a unified user interface was a delicate challenge.

      Solutions:

      • Knowing application specific APIs and functions.
      • Creating and adapting client content strategy.
      • Implementing user display options to reduce clutter.

Key Achievements

  • Enhanced system performance monitoring.
  • Improved resource utilization tracking.
  • Swift error detection and resolution.
  • Strengthened security incident management.
  • Empowered stakeholders with real-time insights.
  • Created a dashboard that solved Client Problem.
  • Managed over 3,000 Log metrics
  • Displayed over 55 key critical operational stats.
