HTML and CSS Emotion Log Form

This project was an artistic challenge to use unique styles to create specific aesthetics and experiment with form designs.

Overview

Emotions are an important part of life that need to be processed for us as humans to function well and find mental relief. This form is an idea on how to help people process their life. Things I'm working on adding to enhance this project are: better specific form validation, more options for the describing feelings, better responsiveness for mobile.
